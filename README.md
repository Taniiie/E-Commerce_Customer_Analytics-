## 📌 Project Overview

This project is an end-to-end data analytics and machine learning solution designed to analyze customer behavior, segment customers, and provide product recommendations.

It simulates real-world business use cases similar to leading e-commerce platforms.

---

## 🎯 Objectives

* Analyze customer purchasing behavior
* Identify high-value customers
* Segment customers using RFM analysis
* Apply machine learning for clustering
* Build a recommendation system
* Create an interactive Power BI dashboard

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Matplotlib, Scikit-learn)
* MLxtend (Association Rules)
* Power BI
* Jupyter Notebook

---

## 📊 Project Workflow

### 🔹 Data Collection

* Used Online Retail dataset (transactional data)

### 🔹 Data Cleaning

* Removed missing CustomerIDs
* Removed duplicates
* Filtered negative quantities (returns)
* Converted date formats
* Created Revenue column

### 🔹 Exploratory Data Analysis (EDA)

* Identified top-selling products
* Analyzed monthly revenue trends
* Country-wise revenue analysis
* Customer purchase behavior

### 🔹 RFM Analysis

* Segmented customers based on:

  * Recency
  * Frequency
  * Monetary value

### 🔹 Customer Segmentation

* Categorized customers into:

  * High Value
  * Loyal
  * At Risk
  * Regular

### 🔹 Machine Learning (Clustering)

* Applied K-Means clustering
* Identified distinct customer groups

### 🔹 Recommendation System

* Built using:

  * Association Rules (Apriori)
  * Cosine Similarity
* Generated product recommendations

### 🔹 Dashboard (Power BI)

* Designed interactive dashboard showing:

  * Revenue trends
  * Customer segmentation
  * Top products
  * Geographic insights
  * Customer clusters

---

## 📈 Key Insights

* High-value customers contribute a major portion of revenue
* A small number of products drive most sales
* Repeat customers are crucial for business growth
* Sales show seasonal trends
* Customer segmentation enables targeted marketing
